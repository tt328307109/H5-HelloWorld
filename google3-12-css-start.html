css也叫做层叠样式表，主要负责网页的样式，css将网页架构和页面显示进行分离

css常见属性

  • color 文字颜色
  • font-size 文字大小
  • width 宽度
  • height 高度
  • backgroud-color 背景颜色
  • backgroud-image 背景图片
  • line-height 行高
  • text-align 文字位置
  • text-indent 对齐方式(首行缩进。)

在css里面经常会去处理跟文本有关的样式，下面看下文本的样式

  • font-family 字体
  • font-size 文字大小
  • font-weight 文字粗细
  • font-style 字体风格
  • line-height 行高

css常见写法-内嵌式写法，css代码写到head标签里面

css常见写法-外链式写法，单独在一个css文件里面去写css代码，写完之后，通过link标签 link href="xxx.css" 这种方式来引入css代码

css常见写法-行内式写法，把css代码写到标签里面。
